Key Features

  • Pin/Unpin Panels: Keep panels permanently visible or allow them to collapse automatically
  • Empty Tabset Hiding: Hide empty tabsets that can appear when populated with content
  • Enhanced Layouts: Support for complex layouts with placeholder tabsets
  • All Original Features: Splitters, drag & drop, popouts, theming, and more
